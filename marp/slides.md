# What is Marp?

**Marp** (Markdown Presentation Ecosystem) is a tool to create beautiful slide decks using Markdown.

## Key Features

- 📝 Write presentations in Markdown
- 🎨 Multiple themes available
- 🚀 Export to HTML, PDF, and PowerPoint
- 🔧 Highly customizable
